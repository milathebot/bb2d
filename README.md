# Bb2D

A cozy browser RPG gift: farming, fishing, forest gathering, home decorating, cats, and a match-3 memory shrine.

## Controls

- Move: WASD / arrow keys
- Interact: E
- Fish: F near pond
- Puzzle: P near memory shrine
- Decorate: B inside home
- Close puzzle: ESC

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
